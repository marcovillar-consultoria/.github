# .github

Repo especial da Organização GitHub `marcovillar-consultoria`. Duas funções, ambas por convenção do próprio GitHub (não configuração explícita):

- **`profile/README.md`** — conteúdo da página de perfil do org (`github.com/marcovillar-consultoria`). GitHub renderiza esse arquivo automaticamente por estar no caminho `profile/`.
- **Raiz deste repo** — defaults org-wide de saúde da comunidade (`SECURITY.md`, `PULL_REQUEST_TEMPLATE.md`, `CONTRIBUTING.md`, `ISSUE_TEMPLATE/`). Qualquer repositório da org sem sua própria cópia herda a versão daqui.

Este repo não tem wiki, issues nem projects habilitados — serve só como container de perfil + defaults, sem uso real para essas features.

SSOT de governança do ecossistema OpenFathom especificamente: [`openfathom-meta`](https://github.com/marcovillar-consultoria/openfathom-meta). SSOT de identidade visual da prática: [`brand`](https://github.com/marcovillar-consultoria/brand). Este repo é independente dos dois — existe pela convenção do GitHub para repos `.github`, não pela topologia de seis repos do OpenFathom.
